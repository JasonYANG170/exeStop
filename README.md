# 应用关闭助手
程序无法关闭？想定时关闭程序？不希望程序偷偷启动？不想让别人在你电脑上使用程序？

快来试试吧！

发布V1.5.0版本

**现项目功能合并至"健康守护"项目**

支持定时关闭程序，支持直接关闭程序，支持禁止程序启动。
# 使用说明
1.找到程序，右键属性，在路径中找到".exe"结尾的程序名称。

2.把名称填入应用关闭助手。

3.点击设定（可不填时间）。

4.选择你需要的功能吧
