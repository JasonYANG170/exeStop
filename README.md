快来试试吧！
# 功能
- ✅ 定时关闭程序
- ✅ 直接关闭程序
- ✅ 禁止程序启动

# 使用说明
1.找到程序，右键属性，在路径中找到".exe"结尾的程序名称。

2.把名称填入应用关闭助手。

3.点击设定（可不填时间）。

4.选择你需要的功能吧
